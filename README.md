# AI UI Prompt Libraries

10 beautifully designed, interactive AI UI Prompt Libraries built with React + TypeScript — each styled in a completely different design system.

## About

This project showcases what a prompt library product UI could look like across 10 distinct visual aesthetics:

1. **Shiuli's Dark Vision** — Glass interface inspired by Apple Vision Pro
2. **Shiuli's Doc Drop** — Editorial database view
3. **Shiuli's Ink Mode** — Figma-style canvas
4. **Shiuli's Moon Era** — Editorial storefront
5. **Shiuli's Sage Edit** — Quiet-luxury editorial layout
6. **Shiuli's Dark Arc** — Dark-mode server layout
7. **Shiuli's Brain Orbit** — Cosmic orbit dashboard
8. **Shiuli's Last Call** — Bento-grid drink bar homepage
9. **Shiuli's Smart Pad** — Tile dashboard
10. **Shiuli's Olive Drop** — Scattered cocktail-menu whiteboard

## Prompts

Every design includes **18 hand-crafted, production-ready prompts** covering both UI design and front-end development workflows:

- Component audits
- Color system generation
- Figma-to-Tailwind conversion
- Dark mode token maps
- Micro-interaction specs
- Accessibility audits
- Onboarding flow outlines
- Storybook scaffolds
- Skeleton loaders
- A/B test variants
- And more

Each prompt is **filterable by category** (Design / Dev / Both), **searchable**, and **copyable with one click**.

## Stack

- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- Vite

## Usage

Fork any component, drop in your own prompts, and ship your own library.
